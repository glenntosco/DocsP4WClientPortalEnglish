# Create Portal Users

To create the users that will allow a 3PL client to access their portal, click on the **client's name** (shown in blue) in the client list. Throughout this documentation, the client **BONLAC** is used as a reference example.

## Client Details & Portal Users Screen

Selecting a client opens the **Client Details** screen along with the **Portal Users** panel. From here the administrator can:

- Create new users
- Assign passwords
- Manage access for that client

> ðŸ“· Screenshot: Client Details screen showing the Portal Users panel.

---

## Adding a User

1. Click **Add**.
2. A user configuration window will open.
3. Enter the required details: **Name**, **Username**, and **Password**.
4. Save the new user.

> ðŸ“· Screenshot: User configuration window with Name, Username, and Password fields.

{% hint style="info" %}
The administrator can add, modify, or delete users for each client from this screen at any time.
{% endhint %}
