# Reports

The Reports module provides visibility into inventory levels and product expiration data. All reports can be filtered and exported to Excel.

| Report | Description |
|---|---|
| [Detailed Inventory Report](inventory-detail.md) | Current inventory by product with full detail |
| [Inventory Snapshots](inventory-snapshots.md) | Historical inventory state at a specific past date |
| [Expiration Report](expiration-report.md) | Products with expiration dates, including already-expired items |
