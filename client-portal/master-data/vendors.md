# Vendors

The Vendors section stores the supplier information used when creating purchase orders.

Vendors can be created **manually** or loaded in bulk via **import**.

---

## Creating a Vendor Manually

When creating a vendor manually, the system will prompt you to enter the following details:

- Vendor name
- Contact information
- Address details
- Any additional required fields
